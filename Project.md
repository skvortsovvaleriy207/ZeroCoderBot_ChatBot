# ZeroCoderBot: Система Мониторинга и Суммаризации Telegram Чатов

## Обзор Проекта

**ZeroCoderBot** — это комплексное решение для автоматического сбора, анализа и визуализации сообщений из Telegram-чатов. Система позволяет пользователям следить за активностью в чатах, получать краткие выжимки (саммари) содержания с помощью искусственного интеллекта (GigaChat) и просматривать статистику через веб-интерфейс.

Проект построен на модульной архитектуре, где каждый компонент выполняет свою специфическую задачу, обеспечивая гибкость и масштабируемость.

## Архитектура и Компоненты

Проект состоит из четырех основных модулей:

### 1. Telethon Script (Сборщик данных)
*   **Директория**: `telethon/`
*   **Назначение**: Подключается к Telegram API как пользователь (UserBot), слушает указанные чаты в реальном времени и сохраняет все входящие сообщения в локальную базу данных SQLite.
*   **Ключевые технологии**: Python, Telethon (MTProto), SQLite.

### 2. Summary Bot (Суммаризатор)
*   **Директория**: `ChatBot/`
*   **Назначение**: Telegram-бот, который взаимодействует с пользователем. Он мониторит базу данных на предмет новых сообщений, отправляет их на обработку в GigaChat для генерации краткого содержания и присылает результат пользователю.
*   **Ключевые технологии**: Python, aiogram (или telebot), GigaChat API, SQLite.

### 3. Flask Dashboard (Веб-аналитика)
*   **Директория**: `flask/`
*   **Назначение**: Веб-приложение для визуализации данных. Предоставляет удобный дашборд со статистикой (количество сообщений, дата последнего анализа) и интерфейс для просмотра истории сообщений.
*   **Ключевые технологии**: Python, Flask, HTML/CSS, SQLite.

### 4. Mini App (Автономное приложение)
*   **Директория**: `miniapp/`
*   **Назначение**: Облегченная, независимая версия веб-интерфейса. Реализует функционал дашборда и просмотра сообщений, работая как самостоятельное Flask-приложение. Может использоваться как Telegram Web App или отдельный сервис.
*   **Ключевые технологии**: Python, Flask, Jinja2, CSS.

## Стек Технологий

*   **Язык программирования**: Python 3.10+
*   **База данных**: SQLite (единое хранилище `messages.db` для всех компонентов).
*   **API и Библиотеки**:
    *   **Telegram API** (через Telethon) для сбора данных.
    *   **Telegram Bot API** для взаимодействия с пользователем.
    *   **GigaChat API** (Sber) для NLP и суммаризации текстов.
    *   **Flask** для веб-разработки.
*   **Фронтенд**: HTML5, CSS3 (переменные CSS, Flexbox/Grid), шрифт Inter.

## Установка и Настройка

### Предварительные требования
1.  **Python 3.10+** установлен в системе.
2.  **Telegram API ID и Hash**: Получить на [my.telegram.org](https://my.telegram.org).
3.  **Bot Token**: Получить у [@BotFather](https://t.me/BotFather).
4.  **GigaChat Credentials**: Получить доступ к API GigaChat.

### 1. Настройка окружения
Создайте файл `.env` в корне проекта и заполните его:
```ini
API_ID=ваш_api_id
API_HASH=ваш_api_hash
SESSION_NAME=my_session
BOT_TOKEN=ваш_токен_бота
GIGACHAT_CREDENTIALS=ваш_ключ_gigachat
```

### 2. Установка зависимостей
Для каждого модуля рекомендуется использовать отдельное виртуальное окружение:

**Telethon:**
```bash
cd telethon
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

**ChatBot:**
```bash
cd ChatBot
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

**Flask / Mini App:**
```bash
cd flask  # или cd miniapp
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

## Запуск Системы

Система требует параллельного запуска компонентов. Рекомендуется использовать `screen`, `tmux` или разные терминалы.

1.  **Сбор данных**: `cd telethon && ./venv/bin/python main.py`
2.  **Бот**: `cd ChatBot && ./venv/bin/python bot.py`
3.  **Веб-интерфейс**: `cd flask && ./venv/bin/python app.py` (доступен по `http://127.0.0.1:5001`)
